# OpenClaw Newspaper 启动说明

## 项目概述

OpenClaw Newspaper 是一个基于 Markdown 的项目管理系统，支持创建项目和版次，自动生成报纸样式的排版。

## 系统架构

### 服务器架构
- **主 HTTP 服务器**（端口 8000）：提供静态文件服务
- **管理 API 服务器**（端口 8001）：处理项目创建和版次创建请求

### 核心组件
- `site/index.html`：主门户页面，展示所有项目
- `site/admin/index.html`：项目管理界面
- `scripts/admin_api.py`：后端 API 服务
- `scripts/markdown_converter.py`：Markdown 转换器
- `projects/`：项目目录，每个项目一个文件夹

## 启动步骤

### 方法一：使用启动脚本（推荐）

```powershell
# 在项目根目录运行
powershell -ExecutionPolicy Bypass -File start_servers.ps1
```

**注意**：此方法会启动两个 PowerShell 窗口，分别运行主服务器和 API 服务器。

### 方法二：手动启动

如果启动脚本无法正常工作，请手动启动：

#### 1. 启动主 HTTP 服务器

```powershell
# 在项目根目录运行
py -m http.server 8000
```

#### 2. 启动管理 API 服务器

在另一个 PowerShell 窗口中运行：

```powershell
# 在项目根目录运行
py scripts/admin_api.py
```

## 验证启动

### 检查端口占用

```powershell
# 检查主服务器（端口 8000）
netstat -ano | findstr :8000

# 检查 API 服务器（端口 8001）
netstat -ano | findstr :8001
```

如果看到 `LISTENING` 状态，说明服务器已成功启动。

### 访问页面

- **主门户页面**：http://localhost:8000/site/index.html
- **项目管理界面**：http://localhost:8000/site/admin/index.html

## 使用流程

### 1. 创建项目

1. 访问 http://localhost:8000/site/admin/index.html
2. 在 Markdown 编辑器中输入项目信息
3. 点击"创建项目"按钮
4. 系统会自动生成项目目录和文件

### 2. 创建版次

1. 在项目管理界面选择"创建版次"标签
2. 选择要添加版次的项目
3. 输入版次信息（标题、类型、内容等）
4. 点击"创建版次"按钮

### 3. 查看项目

1. 访问 http://localhost:8000/site/index.html
2. 点击项目卡片进入项目详情页
3. 在项目详情页中可以看到所有版次

## Markdown 格式

### 项目格式

```markdown
# 项目名称

## 摘要
项目摘要内容...

## 阶段
当前阶段描述...

## 阻塞与问题
阻塞问题描述...

## 下一步计划
下一步计划内容...

## 正文内容
这里是主要的报纸正文内容...
```

### 版次格式

```markdown
# 版次标题

## 类型
快报 / 头版

## 摘要
版次摘要...

## 正文内容
版次的详细内容...
```

## 故障排除

### 问题：端口被占用

**解决方案**：
```powershell
# 查找占用端口的进程
netstat -ano | findstr :8000

# 结束进程（将 <PID> 替换为实际的进程 ID）
taskkill /PID <PID> /F
```

### 问题：API 服务器无法启动

**检查事项**：
1. 确认 Python 环境正常
2. 检查端口 8001 是否被占用
3. 查看终端错误信息

### 问题：页面无法访问

**检查事项**：
1. 确认两个服务器都已启动
2. 检查防火墙设置
3. 尝试使用 `http://127.0.0.1:8000` 替代 `http://localhost:8000`

## 开发说明

### 项目显示逻辑

项目详情页（`projects/{project-name}/index.html`）包含：
- **顶部项目切换栏**：显示所有项目，可快速切换
- **左侧项目摘要栏**：显示项目基本信息（摘要、阶段、阻塞、下一步）
- **右侧版次目录**：显示所有版次，可快速跳转
- **主内容区**：以 iframe 形式嵌入各个版次

### 版次显示逻辑

每个版次包含：
- **版次头部**：类型、标题、时间、摘要
- **版次内容**：以 iframe 形式嵌入独立的 HTML 文件
- **iframe 高度**：根据内容自动调整

### 样式系统

- **颜色方案**：纸质背景色（`#f5f1e8`）、深色文字（`#171717`）、强调色（`#7f1111`）
- **字体**：衬线字体（Source Han Serif SC）用于正文，无衬线字体（Inter）用于 UI
- **布局**：响应式设计，支持桌面和移动端
